If you use the spec or spec_set arguments then only magic methods that exist in the spec will be created.

Attributes and the return value of a MagicMock will also be MagicMocks.

security.create_access_token(data: dict, expires_delta: timedelta | None = None) → str[source]¶

Create a JWT access token.

Parameters:

  • data (dict) – Data to encode in the token (e.g., user ID).

  • expires_delta (timedelta, optional) – Expiration time for the token. Defaults to ACCESS_TOKEN_EXPIRE_MINUTES if not provided.

Returns:

Encoded JWT token.

Return type:

str

security.get_password_hash(password: str) → str[source]¶

Hash a plain password using bcrypt.

Parameters:

password (str) – Plain text password.

Returns:

Hashed password.

Return type:

str

security.verify_password(plain_password: str, hashed_password: str) → bool[source]¶

Verify that a plain password matches a hashed password.

Parameters:

  • plain_password (str) – Password provided by the user.

  • hashed_password (str) – Hashed password stored in the database.

Returns:

True if the password matches, False otherwise.

Return type:

bool
